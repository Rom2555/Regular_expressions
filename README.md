# Regular_expressions — Учебный проект Нетология

> Обработка телефонного справочника с использованием регулярных выражений в Python

Этот проект создан в рамках учебного курса **Нетологии** по изучению **регулярных выражений**.  
Цель — на практике освоить парсинг и форматирование текстовых данных с помощью `re` в Python.

---

## Задача

Преобразовать "сырой" CSV-файл с контактами (`phonebook_raw.csv`) в чистый, структурированный телефонный справочник (`phonebook.csv`), выполнив:

- Разбор и нормализацию ФИО
- Объединение дубликатов контактов
- Форматирование телефонных номеров с помощью **регулярных выражений**
- Заполнение пропущенных данных
- Сортировку по алфавиту

---

## Что используется

- **Python 3**
- Модуль `csv` — чтение и запись данных
- Модуль `re` — регулярные выражения для форматирования телефонов
- Обработка строк и структурирование данных

---

## Как запустить

1. Убедитесь, что у вас установлен **Python 3**
2. Поместите файл `phonebook_raw.csv` в папку проекта
3. Запустите скрипт: `bash python main.py`
4. Программа создаст файл `phonebook.csv` с обработанными данными

---

## Форматирование телефонов

Используется регулярное выражение для приведения номеров к единому виду:

**Из:**
`8(999)123-45-67, +7 999 123 45 67, 8-999-123-45-67 доб.123`

**В:**
`+7(999)123-45-67, +7(999)123-45-67 доб.123`

Реализовано в функции `format_phone_number()` с использованием `re.sub()`.

---

## Объединение дубликатов

Контакты объединяются по ключу **(фамилия, имя)**:
- Если есть два контакта с одинаковыми фамилией и именем — они объединяются
- Пустые поля дополняются данными из другой записи
- Никакие данные не перезаписываются
